Note that in order to run the producer.py with gesture recognition you need to: 

```
pip install -e ./addons/gesture_recognition
```

From the project's root folder and download the hand_landmarker.task from Google's site from: 

```
https://storage.googleapis.com/mediapipe-models/hand_landmarker/hand_landmarker/float16/latest/hand_landmarker.task
```
